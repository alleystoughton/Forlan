Symbols and Strings

This subsection describes modules relating to Forlan symbols and strings. The Sym module provides the abstract type of Forlan symbols. The Str modules provides operations on Forlan strings, i.e., lists of Forlan symbols. The SymSet module provides operations on finite sets of Forlan symbols. The StrSet module provides operations on finite sets of Forlan strings. And the SymRel module provides operations on finite relations on Forlan symbols.
