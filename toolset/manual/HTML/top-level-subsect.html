Basic Types

Identifier Value
type 'a set 'a Set.set
type sym Sym.sym
type str Str.str
type ('a, 'b)rel ('a, 'b)Rel.rel
type sym_rel SymRel.sym_rel
Note that:
  • sym and Sym.sym are equal to Lex.sym;
  • str and Str.str are equal to sym list;
  • ('a, 'b)rel and ('a, 'b)Rel.rel are equal to ('a * 'b)set;
  • sym_rel and SymRel.sym_rel are equal to (sym, sym)rel, which is equal to (sym * sym)set.
