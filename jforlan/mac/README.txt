jforlan/JForlan.app is built from ../JForlan.jar and
../icons/JForlan.icns using /Developer/Applications/Utilities/Jar
Bundler.app:

  In the "Build Information" tab, check the "Use Macintosh Menu Bar"
  option, and specify "JVM Version 1.6+".

  In the "Properties" tab, "Version" and "Get-Info String" should be
  set to the version of JForlan.
